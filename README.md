# Agenda de Contatos

## Objetivo
Desenvolver uma agenda de contatos com as funcionalidades de:

1. Adicionar contato
2. Exbbir Agenda
3. Editar Contato
4. Remover Contato
5. Salvar Agenda

Por fim, ao salvar a agenda de contatos foi feita a criação de um arquivo JSON para marter os dados salvos, possibilitando o acesso posterior.
